# unitone 既知の問題

## この文書の目的

確認済みの不具合、互換性問題、環境依存問題について、症状、再現条件、原因、回避策、解決条件を記録します。事実と仮説を明確に分け、未検証の推測だけで項目を追加しないでください。

追記前にこの文書を検索し、同じ問題がある場合は新規項目を作らず既存項目を更新してください。問題が解決した後も、将来の互換性判断に有用であれば状態、解決条件、関連する変更を更新して記録を残します。

## 状態の目安

- `未解決`: 問題が継続している
- `回避策あり`: 解決前だが確認済みの回避策がある
- `解決済み`: 修正または外部要因の変更により解決した
- `再現せず`: 記録した条件では再現しなくなったが、解決を断定できない

## 原因の確度

- `確認済み`: 証拠または修正によって原因を確認した
- `有力な仮説`: 証拠はあるが原因を確定していない
- `未確認`: 原因をまだ特定していない
